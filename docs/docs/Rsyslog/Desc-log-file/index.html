常见日志文件说明

/var/log/secure

系统安全相关的日志 - 谁登陆了 - 什么时间登陆了 - 通过哪个账号登陆了 - 从哪台主机登陆了 - 密码输入错误了几次

/var/log/messages

这个文件很重要，几乎系统发生的错误信息都会记录在这个文件中，如果系统发生莫名的错误，一定要查看这个文件。
